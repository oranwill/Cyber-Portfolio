# SOC Lab Framework

## Objective
Build a home SOC lab to simulate attacks and analyze them using Splunk.

## Tools Used
Splunk
Sysmon
Kali Linux
Nmap
VirtualBox

## Lab Components
Attacker Machine – Kali Linux
Victim Machine – Windows 10 with Sysmon
SIEM – Splunk

## Detection Goal
Detect network scans using Sysmon Event ID 3 in Splunk.
